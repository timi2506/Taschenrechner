# Taschenrecher
Diese SwiftUI App wurde für meine 10. Klasse Facharbeit zum Thema "KI beim Programmieren" von ChatGPT generiert.

Alle Genutzten Prompts sind unter [Infos.md](https://github.com/timi2506/Taschenrechner/blob/main/Infos.md) zu finden und jedem Prompt ist ein Screenshot angehangen welche unter [Prompt Screenshots](https://github.com/timi2506/Taschenrechner/tree/main/Prompt%20Screenshots) zu finden sind.

Der Komplette ChatGPT Chatverlauf lässt sich unter [diesem Link](https://chatgpt.com/share/696ccee3-9ff0-8011-8c40-1f2a7bf6f1bb) finden, optional auch als [PDF](https://github.com/timi2506/Taschenrechner/blob/main/ChatVerlauf.pdf) oder als [HTML](https://github.com/timi2506/Taschenrechner/blob/main/ChatVerlauf.html)
